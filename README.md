## Detection capabilities

### Memory Integrity Monitor (MIM)

| Attack | Description | Detection |
|--------|-------------|-----------|
| **MINJA** | Memory INJection Attack вҖ” query-only poisoning | Bridging query вҶ’ indication вҶ’ self-write sequence |
| **SpAIware** | Cross-session malware propagation | Content fingerprint repeats across session boundaries |

14 regex patterns. Fires on `high`/`critical` severity only.

### Coordination Signatures (MAST)

Based on [arXiv:2503.13657](https://arxiv.org/abs/2503.13657) вҖ” NeurIPS 2025 Spotlight вҖ” Cohen's Оә=0.88 across 1,642 annotated traces.

```
Category 1 вҖ” Specification     task misinterpretation В· role ambiguity В· poor decomposition
                                duplicate roles В· missing termination condition

Category 2 вҖ” Inter-agent       handoff breakdown В· context loss В· conflicting parallel outputs
                                format mismatch between agents

Category 3 вҖ” Verification      premature termination В· incomplete verification В· incorrect logic
                                missing error recovery В· no feedback loop

Infrastructure                  infinite retry loop В· rate limit cascade В· context overflow
                                API version drift В· framework misuse
```

### Verdict Engine

```
Stage 1 вҖ” Deterministic   cost > $0.10 В· spans > 50 В· any framework_fault В· policy violation
               вҶ“ (if not conclusive)
Stage 2 вҖ” Baseline        3Пғ deviation from per-agent behavioral profile (50-trace window)
               вҶ“ (sampled 20%)
Stage 3 вҖ” LLM Judge       receives summarized trace only вҖ” never raw spans
```

### Content Inspector

15 regex patterns covering:
- Direct instruction injection (`ignore previous instructions`, `[INST]` blocks)
- System/role override (`SYSTEM:`, `you are now`, `you are unrestricted`)
- Jailbreaks (`DAN`, `forget training`, `no rules apply`)
- Exfiltration (`send * to https://`, `forward credentials to`)
- Bypass attempts (`override safety`, `disregard guidelines`)

## Security invariants

These are enforced in code and verified in every gate run:

```
вңҰ  Chronicle is APPEND-ONLY. No UPDATE. No DELETE. Ever.
вңҰ  Signal origin HMAC-verified on every emission, not just the first.
вңҰ  Policy Engine is DEFAULT-DENY. Must be permitted, not just not forbidden.
вңҰ  Interceptor logs EVERY ACTION to Chronicle. Never silent.
вңҰ  New agent runs in RESTRICTED MODE until 50 traces establish baseline.
вңҰ  LLM Judge receives SUMMARIZED TRACE вҖ” never raw spans.
```
